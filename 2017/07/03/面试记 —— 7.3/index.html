如果深复制的对象的属性值本身也是一个对象或数组，对判断这个对象或数组，它的定义是通过{}字面量或new构造的对象才是对象(jQuery.isPlainObject)，因为判断结果的偏差会造成深复制失效。例如：

将lodash和jQuery作对比，当改变了源对象的属性时，目标对象的属性也会被改变，而lodash不会出现这样的情况。正是由于源码中定义深复制的key：value中的value值必须是{}或new定义出来的，其他的会直接浅复制target[ name ] = copy;造成的。
